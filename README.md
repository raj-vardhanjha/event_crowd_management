# Event Crowd Management System

A modern web application for managing events and crowd control, featuring a sleek UI with animations and comprehensive event management capabilities.

## Features

- **Modern User Interface**
  - Responsive design for all devices
  - Smooth animations and transitions
  - Interactive event cards with hover effects
  - Dark theme with gradient accents

- **Event Management**
  - View upcoming and ongoing events
  - Organize new events
  - Track event statistics and registrations
  - Manage event capacity and details

- **User Features**
  - User authentication system
  - Personal dashboard for organized events
  - Event registration system
  - Real-time registration tracking

## Pages

1. **Homepage**
   - Hero section with animated bubbles
   - Featured events showcase
   - Website features section
   - About section

2. **Events Page**
   - Upcoming events slider
   - Ongoing events display
   - Event filtering options
   - Registration functionality

3. **My Events Dashboard**
   - Event statistics
   - Registration tracking
   - Event management tools
   - Status updates

4. **Event Organization**
   - Comprehensive event creation form
   - Multiple event types support
   - Venue/platform selection
   - Capacity management

## Tech Stack

- HTML5
- CSS3 (with modern features like Grid, Flexbox, and CSS Variables)
- Vanilla JavaScript
- Font Awesome for icons
- Google Fonts (Poppins)

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/event-crowd-management.git
```

2. Open the project:
```bash
cd event-crowd-management
```

3. Launch the application:
- Open `index.html` in your web browser
- Or use a local server:
  ```bash
  python -m http.server 8000
  # Then visit http://localhost:8000
  ```
